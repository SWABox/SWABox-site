
# 设计禁止清单：
- 紫色。靛蓝色渐变（#6366F1 #8B5CF6）
- 纯平背景色（必须有噪点或渐变）
- Tailwind 默认色板
- Hero+三卡片布局
- 完美居中对齐
- 等宽多栏（必须不对称）
- 高深的专业名词和无意义的空话
- Lorem Ipsum 占位文本
- 被动时态和长句
- Shadcn/Material UI 默认组件（必须深度定制）
- Emoji作为功能图标
- 线性动画 ease-in-out
