# SWABox 开发与使用文档

> Smart Whiteboard Administrator · 电教委白板管理工具箱

SWABox 专为教育场景下的智能白板批量运维而生，集成优化、修复、常用软件安装与应急维护工具。

## 🚀 快速开始

### 最低运行环境

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10 / 11 (64位) |
| 权限 | 需管理员权限运行 |
| 依赖 | 无额外运行时（绿色版） |
| 内存 | ≥ 512MB 可用内存 |

### 下载与安装

1. 前往 [下载页](../download) 获取最新 Windows 版
2. 解压后右键 `SWABox.exe` → **以管理员身份运行**
3. 首次运行如被杀毒软件拦截，请加入白名单（开源可审计）

> ⚠️ **注意**：本工具仅面向设备管理用途，请勿用于未授权设备。
