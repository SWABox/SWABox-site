隐私政策

最后更新日期：2026年5月20日

欢迎访问 SWABox 官方网站（以下简称“本站”）。本站由开发者个人维护，托管于 Cloudflare Pages。本政策旨在说明我们在运营过程中如何处理与您相关的信息。

1. 我们收集的信息

本站是一个纯静态网站，本身不具备后端数据库，因此我们不直接收集或存储您的个人身份信息。

  • 本站没有用户注册、登录功能。
  • 没有评论、留言板等交互表单。
  • 本站自身不设置 Cookie，也不使用本地存储（LocalStorage）追踪用户。
  • 不嵌入第三方广告或营销追踪脚本。

2. 托管平台自动记录的技术信息

为了提供快速的访问体验和必要的安全防护，本站全站接入了 Cloudflare 服务（包括 DNS、CDN、SSL 及邮件路由）。当您访问本站时，Cloudflare 作为服务提供商，可能会自动接收并记录您的设备发出的技术信息，包括但不限于：

  • IP 地址（用于分配最近的 CDN 节点及安全防护）。
  • 请求时间与访问的页面 URL。
  • 浏览器 User-Agent（浏览器类型及版本）。
  • Referrer（引荐来源页面）。

这些信息主要用于网络传输、DDoS 防护和性能优化。Cloudflare 会按照其隐私政策处理这些数据。作为站点维护者，我们无法直接访问能精准识别您个人身份的未脱敏原始日志。

关于 Cloudflare 的数据处理详情，请参阅：Cloudflare 隐私政策。

3. 网站分析服务

为了了解网站的访问情况（如独立访客数、热门页面），本站启用了 Cloudflare Web Analytics。该服务具有以下特点：

  • 不使用 Cookie，不会在您的设备上存储任何标识符。
  • 不跟踪跨站行为，数据仅在单次访问会话内聚合。
  • 所有数据均为匿名聚合数据，我们无法通过该分析工具反查特定个人的浏览行为。

4. 通信与邮件处理

若您通过邮件联系我们，或通过 GitHub 与我们互动，相关通信数据将由以下服务处理：

  • Cloudflare Email Routing：本站域名下的邮箱（如 content@swabox.cc.cd）通过 Cloudflare 邮件路由功能进行转发，邮件内容会被临时处理以完成投递。
  • GitHub：若您在 GitHub Issues 中交流，数据受 GitHub 隐私政策保护。

您可以通过 content@swabox.cc.cd 与我们取得联系。

5. 第三方链接

本站包含指向外部网站（如 GitHub 仓库、软件下载源）的链接。这些网站拥有独立的隐私政策，我们对其内容或数据处理实践不承担任何责任。建议您在离开本站后，阅读相关网站的隐私声明。

6. 儿童隐私

本站主要服务于教育技术设备管理，不面向 13 岁以下儿童，也不会故意收集儿童的个人信息。

7. 政策更新

我们可能会不时更新本隐私政策以适应法律法规或服务变更。更新后的政策将发布在本页面，并立即生效。建议您定期查阅。

8. 联系我们

如果您对本隐私政策有任何疑问、意见或希望行使相关数据权利，请通过以下方式联系我们：

  • 电子邮件：content@swabox.cc.cd
  • GitHub Issues：项目 Issues 页面
