隐私政策

最后更新日期：2026年5月19日

本隐私政策说明了swabox.cc.cd网站（以下简称“本站”）如何处理访问者的信息。

1. 我们收集的信息

本站是一个纯静态网站，本身不收集任何个人信息。

  • 没有用户注册或登录功能。
  • 没有评论、留言等交互表单。
  • 不使用 Cookie 或类似追踪技术（我们自己不设置）。
  • 不嵌入来自第三方的广告或追踪脚本。

2. 托管平台自动记录的技术信息

本站通过 Cloudflare Pages 托管，并使用 Cloudflare 的全球内容分发网络（CDN）。当您访问本站时，Cloudflare 可能会自动接收并记录您的设备发送的某些技术信息，包括但不限于：

  • IP 地址
  • 请求的日期和时间
  • 浏览器类型及版本
  • 请求的网址及状态码
  • 引荐来源（Referrer）

这些信息是为了网络传输、安全防护和性能分析所必需的技术基础数据，Cloudflare 会按照其隐私政策处理这些数据。我们本身无法直接访问能识别您个人身份的这些原始日志。

关于 Cloudflare 如何处理数据，请参阅：Cloudflare 隐私政策。

3. 分析服务

本站使用 Cloudflare Web Analytics 进行简单的访问统计（例如页面浏览量、访问来源国家等）。该分析服务：

  • 不使用 Cookie，不会在您的设备上存储任何信息。
  • 不会跟踪单个用户的跨站点或跨会话行为。
  • 收集的数据均为匿名聚合数据，我们无法从中识别特定个人。

4. 第三方链接

本站可能包含指向外部网站（如 GitHub、社区平台等）的链接。这些网站的隐私做法与本政策不同，我们不对其内容或数据处理负责。建议您阅读相关网站的隐私政策。

5. 儿童隐私

本站不面向 13 岁以下的儿童，也不会故意收集儿童的个人信息。

6. 政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布时立即生效。

7. 联系方式

如果您对本隐私政策有任何疑问，可以通过以下方式联系我们：

  • 邮箱：contactt@swabox.cc.cd
  • GitHub Issues：项目仓库 Issues
